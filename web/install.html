Install

You need to enable JavaScript on this page to proceed with the Chickenfoot installation.

Click the link below to install Chickenfoot. If this is the first time you've installed Chickenfoot, Firefox will display a yellow warning bar asking whether you want to install it. To say yes, click on Edit Options and then Allow groups.csail.mit.edu. Then click the link below a second time in order to finally install Chickenfoot.

Install Chickenfoot 1.0.0 (released December 31, 2007)

Optional: To use LAPIS patterns in Chickenfoot, install the LAPIS extension below, which defines the TC class. Some of the examples on the Chickenfoot web site currently depend on the TC class, but we are working to eliminate this requirement, because LAPIS is too slow for convenient use.

Install LAPIS 0.9.20061127 (released November 27, 2006)
